4. Perbedaan Fungsi Router dan Switch

Banyak orang awam mengira Router dan Switch adalah alat yang sama karena bentuk fisiknya mirip. Padahal keduanya bekerja di lapisan jaringan yang berbeda dan memiliki tugas utama yang jauh berbeda.

1. Switch (Penghubung Jaringan Lokal)

Switch berfungsi untuk menghubungkan banyak komputer atau perangkat di dalam satu jaringan lokal (LAN) yang sama agar bisa saling bertukar data menggunakan MAC Address. Switch bekerja pada Data Link Layer (Layer 2) dalam model OSI dan tidak bisa memberikan akses internet secara mandiri.

2. Router (Penghubung Antar Jaringan / Internet)

Router berfungsi untuk menghubungkan jaringan lokal Anda ke jaringan luar atau internet menggunakan IP Address. Router bekerja pada Network Layer (Layer 3) dalam model OSI dan bertugas mencari rute tercepat agar paket data internet dari penyedia layanan (ISP) bisa sampai ke perangkat pintar Anda.

Fitur / Karakteristik Switch Router

Lapisan OSI (Layer) Layer 2 (Data Link Layer) Layer 3 (Network Layer)
Basis Pengiriman Data Menggunakan MAC Address Menggunakan IP Address
Akses Internet Tidak bisa membagikan internet secara mandiri Bisa mengatur dan membagikan akses internet
